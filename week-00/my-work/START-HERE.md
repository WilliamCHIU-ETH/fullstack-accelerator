# 從這裡開始

歡迎來到 Week 0！這個資料夾是你寫 code 的地方。

## 第一步：環境確認

```bash
# 在 fullstack-accelerator/ 根目錄執行
./scripts/verify-env.sh
```

確認全部綠燈後繼續。

## 第二步：Claude Code 初體驗

在這個資料夾裡啟動 Claude Code：

```bash
claude
```

試試以下任務：
1. 「幫我建立一個 hello.js，裡面印出 Hello World」
2. 用 `node hello.js` 跑起來
3. 「把 hello.js 改成問使用者名字，然後用名字打招呼」
4. 再次 `node hello.js` 測試
5. 在 Claude Code 裡輸入 `/help` 看指令列表

## 第三步：開始猜數字遊戲

回到 `week-00/` 資料夾：

```bash
cd ..
```

1. 閱讀 `BUILDERS-LOG.md`
2. 用 `./walk.sh list` 查看步驟
3. 在這個 `my-work/` 資料夾裡建立你的 `game.js`
4. 一步一步完成每個 Step

## 重要提醒

- 你的 code 寫在這裡（`my-work/`）
- 參考 code 用 `./walk.sh look N` 查看
- 測試用 `npm test`（在 `week-00/` 目錄下執行）
